# Safeclean

[ ABOUT ]

  Safeclean is a simple script for Arch Linux with the function of 
    removing the trash in a safe way, with no chance of recovering files.

[ VERSION ]

  Current: 1.0.1

[ DEPENDENCIES ]

  * Linux System
  * wipe
  * shred

[ INSTALL ]

 # make install

[UNINSTALL]

  # make uninstall
 
[ USAGE ]

  For information on how to use it, run the command below:

  $ safeclean -h
